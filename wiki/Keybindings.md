# Keybindings

All keybindings are defined in `hypr/keybinds.lua`.

## Launch & Windows

| Key | Action |
|-----|--------|
| `SUPER`+`Return` | Open terminal (kitty/wezterm) |
| `SUPER`+`Space` | App launcher (rofi) |
| `SUPER`+`Q` | Close active window |
| `SUPER`+`F` | Toggle fullscreen |
| `SUPER`+`V` | Toggle floating |
| `SUPER`+`J` | Toggle split |
| `SUPER`+`P` | Pin window |

## Navigation

| Key | Action |
|-----|--------|
| `SUPER`+`h/j/k/l` | Focus window |
| `SUPER`+`SHIFT`+`h/j/k/l` | Move window |
| `SUPER`+`1-9` | Switch workspace |
| `SUPER`+`SHIFT`+`1-9` | Move window to workspace |
| `SUPER`+`TAB` | Cycle workspaces |
| `SUPER`+`SHIFT`+`TAB` | Cycle windows |

## Screenshots & Recording

| Key | Action |
|-----|--------|
| `Print` | Full screenshot → `~/Pictures/Screenshots/Fullscreen/` + clipboard |
| `SUPER`+`SHIFT`+`S` | Selection screenshot → `~/Pictures/Screenshots/Freeform/` + clipboard |
| `SUPER`+`R` | Start fullscreen recording |
| `SUPER`+`SHIFT`+`R` | Start region recording |
| `SUPER`+`S` | Stop any active recording |

## Media & Volume

| Key | Action |
|-----|--------|
| `SUPER`+`M` | Toggle media player (play/pause) |
| `SUPER`+`SHIFT`+`M` | Next track |
| `SUPER`+`CTRL`+`M` | Previous track |
| `XF86AudioRaiseVolume` | Volume up |
| `XF86AudioLowerVolume` | Volume down |
| `XF86AudioMute` | Toggle mute |

## Brightness

| Key | Action |
|-----|--------|
| `XF86MonBrightnessUp` | Brightness up |
| `XF86MonBrightnessDown` | Brightness down |

## System

| Key | Action |
|-----|--------|
| `SUPER`+`B` | Toggle waybar |
| `SUPER`+`SHIFT`+`P` | Power menu (rofi) |
| `SUPER`+`Escape` | Exit Hyprland |
| `SUPER`+`SHIFT`+`E` | Reload config |
| `SUPER`+`SHIFT`+`C` | Reload waybar |

## Clipboard

| Key | Action |
|-----|--------|
| `SUPER`+`SHIFT`+`V` | Clipboard history (rofi) |

## Emoji

| Key | Action |
|-----|--------|
| `SUPER`+`.` | Emoji picker (rofi) |
